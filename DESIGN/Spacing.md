# Spacing

OpenSnap uses a small, predictable spacing scale.

| Token | Value | Use |
| --- | ---: | --- |
| `space-1` | 4 | Tight icon/text gaps |
| `space-2` | 8 | Standard control gaps |
| `space-3` | 12 | Small group padding |
| `space-4` | 16 | Default panel padding |
| `space-5` | 24 | Section spacing |
| `space-6` | 32 | Large group separation |
| `space-7` | 48 | Major layout separation |

## Rules

- Start with 8 or 16.
- Use 4 only for tight internal relationships.
- Use 24 or 32 to separate ideas.
- Avoid arbitrary spacing values unless needed for optical correction.
- Prefer native control spacing when using native controls.
