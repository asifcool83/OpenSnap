# Vision

In five years, OpenSnap should be the best free and open-source native macOS window manager.

It should be trusted because it is fast, private, reliable, and simple. It should feel like a missing part of macOS: small enough to understand immediately, strong enough to depend on every day, and clean enough that contributors enjoy maintaining it.

OpenSnap should never become a bloated productivity suite. Its value is focus.

## Five-Year Goals

- A polished native macOS app with excellent keyboard-first window management.
- A stable, tested Window Engine that handles common macOS edge cases gracefully.
- A clear design system rooted in the 60/40 identity.
- A contributor-friendly codebase with clean architecture and documented decisions.
- A privacy-first reputation: no telemetry, analytics, accounts, subscriptions, or cloud dependency.
- A release process that supports signed, notarized builds.
- A community that values quality over feature volume.
