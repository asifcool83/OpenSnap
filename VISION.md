# Vision

In five years, OpenSnap should be the most trusted free and open-source native macOS window manager.

It should feel like a missing part of macOS: simple enough to understand immediately, fast enough to disappear, reliable enough to use every day, and clean enough that contributors enjoy maintaining it.

OpenSnap should not become a bloated productivity suite. Its value is focus.

## Five-Year Direction

- A polished native macOS app with excellent keyboard-first window management.
- A stable, tested Window Engine that handles common macOS edge cases gracefully.
- A clear design system rooted in the 60% / 40% identity.
- A contributor-friendly codebase and repository with clean architecture and documented design, workflow, and decisions.
- A privacy-first reputation: no telemetry, analytics, accounts, subscriptions, or cloud dependency.
- A release process that supports signed, notarized builds.
- A community that values quality over feature volume.
